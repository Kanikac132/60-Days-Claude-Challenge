# Day 28 - Hospital Admission Readiness Simulator

## Overview

For Day 28 of the AI Challenge, I created an interactive **Hospital Admission Readiness Simulator** that evaluates whether a patient is ready for hospital admission based on multiple healthcare workflow factors.

The simulator allows users to configure patient admission details, analyze risks, complete workflow actions, and understand how different factors impact the final admission decision.

## Features Implemented

* Provider and physician information input
* Diagnosis and admission type selection
* Prior Authorization status management
* Admission date configuration
* Initial admission readiness score calculation
* Workflow-based risk improvement actions
* Documentation risk analysis
* Insurance and authorization risk handling
* Bed availability and clinical risk management
* Approval, pending, and appeal scenarios
* Governance Snapshot review
* Final admission decision analysis
* Multiple diagnosis scenario testing

## Technologies Used

* HTML
* CSS
* JavaScript
* AI-assisted development using Claude
