# Day 9 – NutriScope AI Application

## Objective

Build and enhance the NutriScope nutrition analysis application using Claude through an iterative prompt-based workflow.

## Work Completed

* Generated the MVP version of NutriScope.
* Enhanced the application using a second prompt.
* Compared the MVP and enhanced versions.
* Captured screenshots of both versions.
* Saved both HTML applications.
* Documented comparison notes and key learnings.
* Organized all project files inside the `Day9` folder.

## Files Included

* MVP NutriScope HTML
* Enhanced NutriScope HTML
* Application Screenshots
* Comparison Notes
* Learning Summary

## Key Learnings

* Iterative prompting significantly improves AI-generated applications.
* Structured prompts help produce more organized and feature-rich outputs.
* Comparing different AI-generated versions provides valuable insights into application refinement and prompt engineering techniques.
