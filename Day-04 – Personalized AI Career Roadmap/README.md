# Day 4 – Personalized AI Career Roadmap

## Overview

On Day 4, I explored how AI can be used as a career planning assistant by generating a personalized career roadmap using Claude.

## What I Did

* Created a structured prompt for career roadmap generation.
* Answered four career-related questions.
* Generated a personalized one-page career roadmap.
* Reviewed the roadmap to identify strengths, skill gaps, learning priorities, recommended projects, networking strategies, and monthly milestones.

## Key Learnings

* AI can support strategic career planning when provided with clear context.
* Structured prompts produce more personalized and actionable outputs.
* Identifying skill gaps helps prioritize learning efficiently.
* Breaking long-term goals into monthly milestones makes progress more achievable.

## Outcome

Successfully generated a personalized AI-powered career roadmap and documented the learning process as part of the **60 Days Claude Challenge**.
