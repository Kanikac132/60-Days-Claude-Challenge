# Day 26 - Prior Authorization Workflow Simulator

## Overview

For Day 26 of the AI Challenge, I created a **Prior Authorization Workflow Simulator**, an interactive HTML application that demonstrates how a healthcare prior authorization process works.

The simulator allows users to manage a patient scenario, move through different workflow stages, prepare authorization requests, submit documents, and experience different outcomes such as approval, pending, denial, and appeal.

## Features Implemented

* Interactive patient workflow simulation
* Multiple workflow stages
* Patient journey tracking
* Document collection and submission process
* Payer response simulation
* Approval, pending, denial, and appeal scenarios
* Elapsed days tracking
* Educational workflow explanations
* Workflow summary dashboard

## Technologies Used

* HTML
* CSS
* JavaScript
* AI-assisted development using Claude
