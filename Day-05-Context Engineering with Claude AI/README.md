# Day 5 – Context Engineering

## Overview

This task explored the importance of **Context Engineering** by comparing AI responses generated with different levels of contextual information.

## What I Did

* Studied the provided Context Engineering resources.
* Generated an output using **Prompt A** without additional personalized context.
* Started a new conversation and generated another output using **Prompt B** with my own contextual details.
* Compared both responses to observe how context affects relevance, clarity, and personalization.
* Documented the outputs, screenshots, observations, and key learnings.

## Key Learnings

* Context significantly improves the relevance and quality of AI-generated responses.
* Well-defined background information helps AI understand user intent more accurately.
* Small changes in context can lead to more personalized, structured, and actionable outputs.
* Effective Context Engineering is an essential skill for working with Large Language Models (LLMs).
