# Day 02 - Prompt Engineering

As part of my 60 Days Claude Challenge, I explored the fundamentals of Prompt Engineering and learned how better instructions can improve AI-generated results.

## What I Learned:
- Understanding what Prompt Engineering is and why it is important
- The importance of clear instructions, context, and specific requirements
- How to create structured and effective prompts
- How small changes in prompts can improve the quality of AI responses

## Prompt Improvement Practice:

During this task, I experimented with improving AI prompts.

First, I created a simple rough prompt with basic instructions and observed the AI output.

Then, I analyzed the limitations of the rough prompt and created an optimized prompt by adding:
- Clear role and context
- Specific requirements
- Proper structure
- Expected output format

This helped me understand how prompt refinement can make AI responses more accurate, relevant, and useful.

- Claude AI

## Key Takeaway:
A good prompt is not just a question; it is a clear instruction that guides AI to generate better results.
