# Day 3 – Role-Based Prompting

Today, I explored **Role-Based Prompting** and learned how assigning a specific **persona** can significantly improve the quality of AI-generated responses.

To understand this concept, I asked Claude the same question using three different approaches:

* Without any persona
* Founder Persona
* Senior Developer Persona

Although the prompt remained the same, each response reflected the perspective of the assigned persona, providing more relevant and context-aware insights.

### Key Learnings

* Understood the concept of Role-Based Prompting.
* Learned how personas influence AI responses.
* Compared generic and role-specific outputs.
* Discovered that adding the right context leads to more accurate and actionable results.

This exercise reinforced an important Prompt Engineering principle: **the way you frame a prompt is just as important as the question itself.**
