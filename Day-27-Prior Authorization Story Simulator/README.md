# Day 27 - Prior Authorization Story Simulator

## Overview

For Day 27 of the AI Challenge, I developed an interactive **Prior Authorization Story Simulator** that explains the healthcare authorization process through a story-based experience.

The application follows Rahul's healthcare journey and allows users to explore different decisions at each stage while learning how prior authorization workflows operate.

## Features Implemented

* Interactive healthcare story simulation
* Eight structured story chapters
* Decision-based dialogue options
* Multiple story paths based on user choices
* Dynamic progress bar tracking
* Educational explanations for each workflow stage
* Final approval process and key takeaways
* Restart option to explore different outcomes

## Technologies Used

* HTML
* CSS
* JavaScript
* AI-assisted development using Claude
