# Day 21 – Digital Privacy Dashboard

## Overview

As part of the **Claude 60 Days Challenge**, I created an interactive **Digital Privacy Dashboard** using Claude AI. The dashboard provides a visual assessment of an individual's digital footprint, privacy exposure, and potential online risks through multiple analytical sections.

## Dashboard Highlights

* Digital Footprint Score
* Privacy Score Analysis
* Exposure Heatmap
* Company Exposure Ranking
* Digital Twin Profile
* Risk Radar
* Most Valuable Data Assets
* Privacy Improvement Simulator
* Final Privacy Verdict

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Claude AI (Prompt Engineering)

This project enhanced my understanding of digital privacy analytics while strengthening my prompt engineering and frontend development skills. It also demonstrated how AI can be used to quickly generate informative and interactive dashboards for real-world awareness and decision-making.
