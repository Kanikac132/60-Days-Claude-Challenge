# Day 20 – AI Face Puzzle Game

## 📌 Project Overview

Built an interactive **Face Puzzle Game** using Claude AI. The application captures a photo through the user's webcam, converts it into a sliding puzzle, and allows the player to solve it using drag-and-drop or touch interactions.

## 🚀 Features

* Webcam photo capture
* Multiple puzzle difficulty levels (3×3, 4×4, 5×5)
* Drag-and-drop gameplay
* Touch support for mobile devices
* Timer to track completion time
* Move counter
* Leaderboard functionality
* Responsive user interface
* Creative UI enhancements

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Webcam API
* Drag & Drop API

## 📚 What I Learned

* Creating complete applications using AI prompts
* Working with browser webcam permissions
* Handling drag-and-drop and touch events
* Managing game state and puzzle logic
* Implementing timers and move tracking
* Improving user experience with responsive design
* Testing interactive web applications
